# Contract Pricing Optimization & Savings Identification Engine

## Overview
This project simulates a real-world healthcare Group Purchasing Organization (GPO) analytics system. It analyzes line-item purchasing data across hospitals and vendors to identify pricing inefficiencies, contract leakage, and cost-saving opportunities.

## Problem Statement
Healthcare organizations often purchase products at prices higher than negotiated contract rates, leading to significant financial inefficiencies.

## Solution
This project:
- Analyzes pricing data across vendors and hospitals
- Identifies off-contract purchases
- Detects pricing variance
- Estimates potential cost savings

## Key Features
- Line-item pricing analysis
- Contract compliance detection
- Savings opportunity identification
- Vendor and product-level benchmarking

## Tech Stack
- Python (Pandas, NumPy)
- SQL concepts
- Power BI (for dashboards)

## Output
- pricing_analysis.csv → detailed dataset
- pricing_summary.csv → aggregated insights

## Business Impact
- Identified simulated savings opportunities exceeding thousands of dollars
- Improved visibility into pricing inefficiencies
- Enabled data-driven procurement decisions
